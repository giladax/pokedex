# My Project

This project consists of a Python server and a React frontend.

## Prerequisites

- Python 3.8 or later
- Node.js 14 or later

## Running the Project

To start the backend:

1. Navigate to the `backend` directory.
2. Run `python server.py`.

To start the frontend:

1. Navigate to the `frontend` directory.
2. Run `npm install`.
3. Run `npm start`.
